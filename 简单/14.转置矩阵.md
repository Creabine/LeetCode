### 题目

[转置矩阵](https://leetcode-cn.com/problems/transpose-matrix/description/)

### 思路

> 双重遍历。

### 解法

```
/**
 * @param {number[][]} A
 * @return {number[][]}
 */
var transpose = function(A) {
    let result = [];
    for (let i=0;i<A[0].length;i++) {
        let item = [];
        A.forEach(j => {
            item.push(j[i]);
        });
        result.push(item);
    }
    return result;
};
```
