### 题目

[转换成小写字母](https://leetcode-cn.com/problems/to-lower-case/description/)

### 思路

> ASCII

### 解法

```
/**
 * @param {string} str
 * @return {string}
 */
var toLowerCase = function(str) {
    let list = str.split('');
    let newStr = '';
    list.map(i => {
        if(i.charCodeAt(0) >= 65 && i.charCodeAt(0) <= 90) {
            newStr += String.fromCharCode(i.charCodeAt(0)+32);
        } else {
            newStr += i;
        }
    })
    return newStr;
};
```
