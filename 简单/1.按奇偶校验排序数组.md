### 题目

[按奇偶校验排序数组](https://leetcode-cn.com/problems/sort-array-by-parity/description/)

### 思路

> 简单，略。

### 解法

```
/**
 * @param {number[]} A
 * @return {number[]}
 */
var sortArrayByParity = function(A) {
    let a = [];
    let b = [];
    A.map(i => {
        if (i % 2 === 0) {
            a.push(i);
        } else {
            b.push(i);
        }
    })
    return [...a, ...b];
};
```
