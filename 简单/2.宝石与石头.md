### 题目

[宝石与石头](https://leetcode-cn.com/problems/jewels-and-stones/description/)

### 思路

> 字符串转数组，嵌套遍历判断相等。

### 解法

```
/**
 * @param {string} J
 * @param {string} S
 * @return {number}
 */
var numJewelsInStones = function(J, S) {
    const j = J.split("");
    const s = S.split("");
    let count = 0;
    j.map(ij => {
        s.map(is => {
            if (ij === is) {
                count++;
            }
        });
    });
    return count;
};
```
