### 题目

[求众数](https://leetcode-cn.com/problems/majority-element/)

### 思路

> 转map，key为数字，value为出现次数。遍历map获取符合条件的众数。

### 解法

```
/**
 * @param {number[]} nums
 * @return {number}
 */
var majorityElement = function(nums) {
    let map = {};
    nums.forEach(i => {
        map[i] !== undefined ? map[i]++ : map[i]=1;
    });
    let result = 0;
    for (i in map) {
        if (map[i] > nums.length / 2) {
            result = i;
        }
    }
    return +result;
};
```
