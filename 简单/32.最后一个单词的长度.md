### 题目

[最后一个单词的长度](https://leetcode-cn.com/problems/length-of-last-word/)

### 思路

> 简单，略

### 解法

```
/**
 * @param {string} s
 * @return {number}
 */
var lengthOfLastWord = function(s) {
    const list = s.trim().split(' ');
    const lastWord = list[list.length - 1];
    return lastWord.split('').length;
};
```
