### 题目

[只出现一次的数字](https://leetcode-cn.com/problems/single-number/description/)

### 思路

> 遍历生成map，key为数字，value为出现次数，返回只出现一次的。

### 解法

```
/**
 * @param {number[]} nums
 * @return {number}
 */
var singleNumber = function(nums) {
    let map = {};
    nums.forEach(i => {
        map[i] !== undefined ? map[i]++ : map[i]=0;
    });
    let result = null;
    for (i in map) {
        if (map[i] === 0) {
            result = +i;
        }
    }
    return result;
};
```
